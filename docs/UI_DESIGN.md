# UI 设计规范 (UI Design Specification)

> **版本**: 1.0.0 | **日期**: 2026-04-19 | **对应模块**: M-17 Web 前端
> **关联 NFR**: NFR-U01~U04

---

## 1. 设计原则

| 原则 | 说明 | 来源 |
|------|------|------|
| **用户导向** | 操作以用户体验为先，降低认知负荷 | NFR-U01 |
| **即时反馈** | 操作反馈 < 200ms，长时间操作显示进度 | NFR-U02 |
| **容错友好** | 错误有明确提示，危险操作可撤销 | NFR-U03 |
| **可视化配置** | 规则/策略/语义层统一定义，支持 Markdown 编辑 | NFR-U04 |
| **一致性** | 全局统一交互模式、视觉风格、术语 |

## 2. 设计系统

### 2.1 色彩体系

| 角色 | 色值 | 用途 |
|------|------|------|
| Primary | `#1677ff` | 主操作、选中态、链接 |
| Success | `#52c41a` | 健康状态、操作成功 |
| Warning | `#faad14` | 退化状态、需注意 |
| Error | `#ff4d4f` | 异常状态、操作失败 |
| Info | `#1677ff` | 信息提示 |
| Neutral-1 | `#ffffff` | 背景白 |
| Neutral-2 | `#f5f5f5` | 次级背景 |
| Neutral-3 | `#d9d9d9` | 边框、分割 |
| Neutral-4 | `#8c8c8c` | 辅助文字 |
| Neutral-5 | `#595959` | 正文 |
| Neutral-6 | `#262626` | 标题 |
| Neutral-7 | `#1f1f1f` | 深色背景（暗色主题） |

### 2.2 严重级别色彩映射

| 级别 | 颜色 | 图标 | 场景 |
|------|------|------|------|
| DEBUG | `#8c8c8c` 灰 | ○ | 审计日志低级别 |
| INFO | `#1677ff` 蓝 | ● | 常规操作 |
| WARN | `#faad14` 黄 | ▲ | 退化/需注意 |
| ERROR | `#ff4d4f` 红 | ✕ | 操作失败 |
| CRITICAL | `#cf132d` 深红 | ⬛ | 系统严重故障 |

### 2.3 健康状态色彩映射

| 状态 | 颜色 | 图标 | 场景 |
|------|------|------|------|
| Healthy | `#52c41a` 绿 | 🟢 | 工具/Agent/服务正常 |
| Degraded | `#faad14` 黄 | 🟡 | 性能下降/部分可用 |
| Unhealthy | `#ff4d4f` 红 | 🔴 | 不可用/熔断 |
| Unknown | `#8c8c8c` 灰 | ⚪ | 未检测/初始化 |

### 2.4 排版

| 元素 | 字号 | 字重 | 行高 |
|------|------|------|------|
| H1 | 24px | 600 | 1.3 |
| H2 | 20px | 600 | 1.3 |
| H3 | 16px | 600 | 1.5 |
| Body | 14px | 400 | 1.5 |
| Caption | 12px | 400 | 1.5 |
| Code | 13px | 400 (monospace) | 1.6 |

### 2.5 间距

基于 4px 网格系统：
- `xs`: 4px — 紧凑间距（图标与文字）
- `sm`: 8px — 相关元素间距
- `md`: 16px — 卡片内间距
- `lg`: 24px — 区块间距
- `xl`: 32px — 大区块间距
- `2xl`: 48px — 页面级间距

## 4. 核心页面交互规范

### 4.1 智能问答（/qa）

| 交互 | 规范 |
|------|------|
| 输入 | Enter 发送，Shift+Enter 换行，最大 2000 字符 |
| 流式输出 | SSE 逐字显示，打字机效果，光标闪烁 |
| 来源标注 | `[1]` 上标格式，hover 显示来源摘要，点击跳转图谱 |
| 会话管理 | 左侧面板：新建/切换/删除/重命名 |
| 错误处理 | 网络断开显示重连提示，超时显示重试按钮 |
| 历史搜索 | 支持搜索历史消息，高亮匹配 |

### 4.2 知识图谱（/graph）

| 交互 | 规范 |
|------|------|
| 渲染 | D3.js 力导向布局，LOD 分级（< 100 全细节，100-1000 聚合，> 1000 概览） |
| 导航 | 鼠标拖拽平移，滚轮缩放，双击聚焦实体 |
| 选中 | 点击节点高亮关联边，右侧面板显示详情 |
| 搜索 | 顶部搜索框，匹配节点自动居中高亮 |
| 过滤 | 按实体类型/关系类型/时间范围过滤 |
| 路径查询 | 右键两节点 → "查找路径"，高亮最短路径 |

### 4.3 模拟推演（/simulator）

| 交互 | 规范 |
|------|------|
| 场景列表 | 卡片视图，显示状态/进度/时间 |
| 时间控制 | 播放/暂停/步进/速度滑块（1x~100x） |
| 事件注入 | 浮动按钮 → 事件模板选择器 → 参数填写 → 确认注入 |
| 实时更新 | WebSocket 推送，图谱/地图增量更新 |
| 方案对比 | 分屏视图，左右各一个推演方案 |

### 4.4 审计日志（/audit）

| 交互 | 规范 |
|------|------|
| 时间线 | 水平时间轴，可缩放拖拽，事件点按严重级别着色 |
| 列表 | 虚拟滚动，默认 50 条/页，支持无限滚动 |
| 过滤 | 时间范围 + 事件类型 + 操作者 + 严重级别 + 关键词 |
| 详情 | 点击展开详情面板，显示完整上下文和变更对比 |
| 跳转 | 来源标注可点击跳转到图谱/本体/策略对应位置 |

### 4.5 本体管理（/ontology）

| 交互 | 规范 |
|------|------|
| 编辑模式 | 表单模式（默认）+ JSON 模式（高级）双模式切换 |
| 版本对比 | 左右分栏 diff 视图，增删改标记 |
| 回滚 | 确认对话框，显示回滚影响范围 |
| 验证 | 实时校验，错误内联标注 |

## 5. 通用交互模式

### 5.1 加载状态

| 场景 | 组件 | 延迟阈值 |
|------|------|---------|
| 页面加载 | Skeleton 骨架屏 | > 300ms |
| 按钮操作 | Button loading 态 | > 200ms |
| 列表加载 | Spin + 空状态 | > 500ms |
| 图谱加载 | 进度条 + 闪烁占位 | > 1s |

### 5.2 错误处理

| 错误类型 | 处理方式 |
|----------|---------|
| 网络错误 | 全局 Toast "网络连接失败"，自动重试 3 次 |
| 401 未认证 | 跳转登录页 |
| 403 无权限 | Toast "无权限执行此操作" |
| 404 不存在 | 空状态页面，引导返回 |
| 500 服务器错误 | 错误边界页面，支持刷新/反馈 |
| 表单验证 | 字段级红色标注 + 错误提示文字 |

### 5.3 危险操作

| 操作 | 确认方式 |
|------|---------|
| 删除工作空间 | 二次确认弹窗 + 输入空间名称确认 |
| 回滚版本 | 二次确认弹窗 + 显示影响范围 |
| 注销工具 | 二次确认弹窗 |
| 策略变更 | 预览影响 → 确认 → 等待 5s → 生效 |

### 5.4 空状态

| 场景 | 提示 | 操作引导 |
|------|------|---------|
| 无工作空间 | "还没有工作空间" | "创建第一个工作空间" |
| 无问答会话 | "开始一段新对话" | "输入你的问题" |
| 图谱无数据 | "图谱还是空的" | "导入数据" / "运行模拟器" |
| 审计日志空 | "暂无审计记录" | — |

## 6. 无障碍设计

| 要求 | 实现 |
|------|------|
| 键盘导航 | 所有交互元素可通过 Tab 聚焦，Enter/Space 激活 |
| 颜色对比度 | WCAG AA 标准（4.5:1 正文，3:1 大文字） |
| 屏幕阅读器 | aria-label、role、aria-live 区域 |
| 焦点管理 | 路由切换后焦点管理，弹窗焦点陷阱 |
| 减弱动画 | prefers-reduced-motion 媒体查询 |
