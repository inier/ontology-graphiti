# ODAP 前端移动优先设计补充说明

> **版本**: 1.0.0 | **日期**: 2026-04-17
> **状态**: 设计中 | **优先级**: P0

---

## 1. 设计核心原则

### 1.1 移动优先策略

采用 **从小到大（Mobile First）** 的渐进增强设计：

1. **核心优先**：先确保移动端的核心功能完整可用
2. **渐进增强**：平板、桌面端在移动端基础上逐步增加功能和布局复杂度
3. **体验一致**：所有设备保持一致的操作逻辑和视觉风格

### 1.2 响应式断点

| 设备 | 断点 | 布局 | 侧边栏 | 内容列数 |
|------|------|------|--------|----------|
| 手机 | < 576px | 单栏全屏 | 底部 Tab 导航 | 1 列 |
| 平板竖屏 | 576-768px | 单栏+浮动面板 | 可收起侧边栏 | 1-2 列 |
| 平板横屏 | 768-1024px | 双栏布局 | 可折叠侧边栏 | 2 列 |
| 笔记本 | 1024-1440px | 标准两栏布局 | 固定侧边栏 | 2-3 列 |
| 桌面 PC | 1440-1920px | 宽松两栏布局 | 固定侧边栏 | 3 列 |
| 大屏 | ≥ 1920px | 三栏布局 | 固定侧边栏 | 3+ 列 |

## 3. 组件响应式规范

### 3.1 导航组件

| 设备 | 导航类型 | 实现方式 |
|------|----------|----------|
| 手机 | 底部 TabBar | 固定底部，最多 5 个 |
| 平板 | 可收起侧边栏 | 折叠按钮 + 滑出 |
| 桌面 | 固定侧边栏 | 常驻显示，可折叠 |

### 3.2 表格/列表组件

| 设备 | 展示方式 |
|------|----------|
| 手机 | 卡片式列表，每项一行关键信息 |
| 平板 | 紧凑表格，可横向滚动 |
| 桌面 | 完整表格，支持排序筛选 |

### 3.3 图谱组件

| 设备 | 布局 | 详情面板 |
|------|------|----------|
| 手机 | 全屏画布 + 浮动操作按钮 | 底部弹出 Drawer |
| 平板 | 分屏（左图右详情） | 右侧固定面板 |
| 桌面 | 分屏 + 工具栏 | 右侧固定面板 + 额外工具栏 |

### 3.4 地图组件

| 设备 | 布局 | 控件位置 |
|------|------|----------|
| 手机 | 全屏地图 + 浮动按钮 | 右下角浮动 |
| 平板 | 分屏 | 左侧面板 |
| 桌面 | 分屏 + 全工具栏 | 左侧面板 + 顶部工具栏 |

### 3.5 表单组件

| 设备 | 布局 |
|------|------|
| 手机 | 单列堆叠，所有字段垂直排列 |
| 平板 | 双列，部分字段并排 |
| 桌面 | 多列 + 高级选项展开 |

## 4. 交互规范

### 4.1 触摸交互（移动端）

| 交互 | 手势 | 反馈 |
|------|------|------|
| 选择 | 单指点击 | 高亮 + 振动 |
| 拖拽 | 长按 + 拖动 | 阴影 + 缩放 |
| 缩放 | 双指捏合 | 实时缩放 |
| 滑动 | 水平滑动 | 切换 Tab/轮播 |

### 4.2 鼠标交互（桌面端）

| 交互 | 方式 | 反馈 |
|------|------|------|
| 选择 | 单击 | 高亮 |
| 多选 | Ctrl/Shift + 单击 | 多选高亮 |
| 拖拽 | 按住拖动 | 实时预览 |
| 缩放 | 滚轮 | 实时缩放 |
| 右键 | 右键点击 | 上下文菜单 |

## 5. 国际化规范

### 5.1 界面文本

所有界面文本使用 i18n key，不使用硬编码：

| 场景 | 示例 |
|------|------|
| 菜单 | `menu.ontology` → "本体图谱" / "Ontology" |
| 按钮 | `button.submit` → "提交" / "Submit" |
| 提示 | `message.success` → "操作成功" / "Success" |
| 错误 | `error.network` → "网络错误" / "Network Error" |

### 5.2 数字/日期格式化

| 类型 | 中文 (zh-CN) | 英文 (en-US) |
|------|--------------|--------------|
| 数字 | 1,234.56 | 1,234.56 |
| 货币 | ¥1,234.56 | $1,234.56 |
| 日期 | 2026年4月17日 | April 17, 2026 |
| 时间 | 14:30:45 | 2:30:45 PM |
| 百分比 | 78.5% | 78.5% |

### 5.3 RTL 支持预留

界面布局预留 RTL（从右到左）支持：

```css
/* 使用 logical properties */
margin-inline-start: 12px;
padding-inline-end: 16px;
```

## 6. 性能目标

| 指标 | 移动端 | 桌面端 |
|------|--------|--------|
| 首屏加载 | < 3s (3G) | < 1.5s |
| 交互响应 | < 100ms | < 50ms |
| 帧率 | 30fps 最低 | 60fps |
| 包体积 | < 300KB gzip | < 500KB gzip |
