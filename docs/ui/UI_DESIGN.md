# ODAP 前端 UI 设计稿

> **版本**: 1.0.0 | **日期**: 2026-04-16
> **状态**: 设计中 | **优先级**: P0

---

## 1. 设计原则

### 1.1 核心原则

| 原则 | 描述 |
|------|------|
| **本体驱动** | 界面以本体图谱为核心，展示实体、关系、事件 |
| **实时性** | WebSocket 实时推送，数据秒级更新 |
| **可观测性** | 推演过程可视化，决策链路可追溯 |
| **多视图** | 同一数据支持图谱/时间线/地图多种视图 |
| **响应式** | 适配桌面端，支持大屏展示 |

### 1.2 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 19 | UI 框架 |
| TypeScript | 5.x | 类型安全 |
| Ant Design | 6.x | 组件库 |
| AntV G6 | latest | 图谱可视化 |
| ECharts | latest | 图表 |
| Leaflet.js | latest | 地理地图 |
| React Router | 7.x | 路由 |

## 11. 组件库

### 11.1 核心组件

| 组件 | 说明 | 规格 |
|------|------|------|
| StatCard | 统计卡片 | 数字 + 标签 + 趋势箭头 |
| GraphCanvas | 图谱画布 | AntV G6 封装 |
| TimelineView | 时间线视图 | 水平滚动 + 事件卡片 |
| MapView | 地图视图 | Leaflet 封装 |
| EntityPanel | 实体详情面板 | 属性表格 + 操作按钮 |
| EventCard | 事件卡片 | 时间 + 描述 + 参与者 |
| VersionDiff | 版本对比 | 左右对比表格 |
| SimulatorConsole | 推演控制台 | 参数表单 + 实时图表 |

### 11.2 组件状态

| 状态 | 样式 |
|------|------|
| Default | 正常展示 |
| Hover | 轻微阴影，颜色加深 |
| Active | 主色调边框 |
| Disabled | 50% 透明度 |
| Loading | Skeleton 占位 |
