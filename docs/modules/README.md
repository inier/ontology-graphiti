# 模块设计文档索引

本文档索引了领域上情报分析与打击决策系统的所有模块设计文档。

## 模块概览

| 模块 | 路径 | 描述 | 优先级 |
|------|------|------|--------|
| **OpenHarness 桥接** | `openharness_bridge/DESIGN.md` | 连接 OpenHarness 与领域业务逻辑 | P0 |
| **Graphiti 客户端** | `graphiti_client/DESIGN.md` | 双时态知识图谱交互 | P0 |
| **OPA 策略管理** | `opa_policy/DESIGN.md` | 策略治理与权限控制 | P0 |
| **Swarm 编排** | `swarm_orchestrator/DESIGN.md` | 多 Agent 协同编排 | P0 |
| **Skills 领域工具** | `skills/DESIGN.md` | 领域领域技能集 | P0 |
| **Hook 系统** | `hook_system/DESIGN.md` | 生命周期钩子与可扩展性核心 | P1 |
| **MCP 协议集成** | `mcp_protocol/DESIGN.md` | 外部系统集成标准协议 | P1 |
| **权限校验** | `permission_checker/DESIGN.md` | OPA 统一权限校验引擎 | P1 |
| **模拟推演引擎** | `simulation_engine/DESIGN.md` | 沙箱隔离的领域模拟推演 | P1 |
| **本体管理** | `ontology/DESIGN.md` | 领域本体模型定义 | P1 |
| **可视化** | `visualization/DESIGN.md` | 态势可视化展示 | P1 |
| **决策推荐** | `decision_recommendation/DESIGN.md` | 决策支持与方案推荐 | P1 |
