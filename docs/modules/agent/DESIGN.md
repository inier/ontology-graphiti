# Agent 模块 - 设计文档

> **优先级**: P0 | **相关 ADR**: ADR-004, ADR-005, ADR-006

**版本**: 1.0.0 | **日期**: 2026-04-16

---

## 1. 模块概述

Agent 模块是系统的**智能决策中枢**，负责协调各组件完成复杂任务。

**核心职责**:
- OODA 循环执行
- 多 Agent 协同
- 任务规划与执行

---

## 2. 核心组件

### 2.1 Agent 类型

| Agent | 职责 | 输入 | 输出 |
|-------|------|------|------|
| Intelligence Agent | 情报收集与分析 | 传感器数据、查询 | 情报报告 |
| Commander Agent | 决策制定 | 情报、理解结果 | 决策建议 |
| Operations Agent | 执行决策 | 决策指令 | 执行结果 |

### 2.2 架构分层

```
┌─────────────────────────────────────────┐
│           DomainSwarm (编排器)             │
│  - 多 Agent 协同                         │
│  - 任务分配                             │
└─────────────────┬───────────────────────┘
                  │
    ┌─────────────┼─────────────┐
    ▼             ▼             ▼
┌────────┐  ┌────────┐  ┌────────┐
│Intel   │  │Commander│  │Operations│
│Agent   │  │Agent   │  │Agent    │
└────────┘  └────────┘  └────────┘
```
