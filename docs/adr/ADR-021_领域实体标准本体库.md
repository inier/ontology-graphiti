# ADR-021: 领域实体标准本体库

> **来源**: `docs/ARCHITECTURE.md` 第 17 章

**状态**: 已接受 | **日期**: 2026-04-16

---

## 1. 上下文

需要基础的领域实体本体，包括各实体的属性，提供中英文说明。

---

## 2. 决策

构建**领域实体标准本体库**（DomainEntityOntology）。

### 2.1 基础实体

| 实体类型 | 说明 |
|---------|------|
| Unit | 单元 |
| Equipment | 装备 |
| Location | 位置 |
| Weather | 气象 |
| Organization | 组织 |

### 2.2 实体属性

| 属性类型 | 说明 |
|---------|------|
| 基础属性 | 名称、位置、状态 |
| 统计属性 | 数量、能力值 |
| 关系属性 | 指挥、部署、位于 |

---

## 3. 后果

| 正面 | 负面 |
|------|------|
| ✅ 快速启动，无需从零构建 | ❌ 可能需要针对特定领域扩展 |
| ✅ 术语统一，便于集成 | ❌ 需要持续维护 |
| ✅ 中英文支持 | |

---

## 4. 可逆性

**低**。本体是系统核心，变更成本高。
