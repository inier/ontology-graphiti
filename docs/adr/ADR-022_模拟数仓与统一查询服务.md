# ADR-022: 模拟数仓与统一查询服务

> **来源**: `docs/ARCHITECTURE.md` 第 17 章

**状态**: 已接受 | **日期**: 2026-04-16

---

## 1. 上下文

系统需要模拟数据仓库，提供：
- 历史数据查询
- 跨数据源聚合
- 报表生成

---

## 2. 决策

构建**模拟数仓**与**统一查询服务**。

### 2.1 架构

```
┌─────────────────────────────────────────────────────────────┐
│                   QueryService                                   │
├─────────────────────────────────────────────────────────────┤
│  SQL Parser │ Query Planner │ Result Aggregator             │
└─────────────────────────────────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────────────┐
│                   Simulated Warehouse                            │
│  - Mock Data Store                                          │
│  - Historical Snapshots                                     │
└─────────────────────────────────────────────────────────────┘
```

---

## 3. 后果

| 正面 | 负面 |
|------|------|
| ✅ 无需真实数据即可演示 | ❌ 数据真实性有限 |
| ✅ 易于构造边界条件 | ❌ 可能误导决策 |
| ✅ 支持离线演示 | |

---

## 4. 可逆性

**高**。可替换为真实数据仓库。
