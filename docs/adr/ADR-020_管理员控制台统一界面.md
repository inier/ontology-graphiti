# ADR-020: 管理员控制台统一界面

> **来源**: `docs/ARCHITECTURE.md` 第 17 章

**状态**: 已接受 | **日期**: 2026-04-16

---

## 1. 上下文

系统需要管理员控制台，用于：
- 系统配置
- 用户管理
- 监控告警
- 日志查看

需要统一的界面。

---

## 2. 决策

构建**管理员控制台**，基于 React + Ant Design。

### 2.1 功能模块

| 模块 | 功能 |
|------|------|
| 系统配置 | 参数配置、热更新 |
| 用户管理 | 角色、权限 |
| 监控告警 | 指标、告警规则 |
| 日志查看 | 操作日志、审计日志 |

---

## 3. 后果

| 正面 | 负面 |
|------|------|
| ✅ 统一管理入口 | ❌ 开发工作量 |
| ✅ 操作便捷 | ❌ 需要权限控制 |
| ✅ 实时监控 | |

---

## 4. 可逆性

**中**。可拆分为独立后台。
