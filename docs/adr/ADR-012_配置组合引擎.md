# ADR-012: 配置组合引擎

> **来源**: `docs/ARCHITECTURE.md` 第 17 章

**状态**: 已接受 | **日期**: 2026-04-16

---

## 1. 上下文

系统中存在多种配置项，需要：
- 灵活组合不同配置
- 支持多环境（开发/测试/生产）
- 动态切换配置

---

## 2. 决策

构建**配置组合引擎**，支持配置叠加和覆盖。

### 2.1 配置优先级

```
环境变量 > 命令行参数 > 用户配置 > 默认配置
```

### 2.2 配置结构

```python
ConfigManager:
  - load_config(env: str)      # 加载环境配置
  - get(key: str, default)     # 获取配置
  - set(key: str, value)       # 设置配置
  - reload()                   # 热重载
```

---

## 3. 后果

| 正面 | 负面 |
|------|------|
| ✅ 灵活的配置组合 | ❌ 配置项多时复杂度增加 |
| ✅ 支持多环境 | ❌ 需要配置验证机制 |
| ✅ 动态切换 | |

---

## 4. 可逆性

**高**。可简化为静态配置。
